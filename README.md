# Legend Cloud Portfolio

A highly available AWS portfolio application built to practice cloud architecture concepts for AWS SAA-C03.

## Current Version

Local static web application using HTML and CSS.

## Final Architecture Goal

Internet → Application Load Balancer → Auto Scaling Group → EC2 Instances → CloudWatch

## AWS Services Practiced

- EC2
- Application Load Balancer
- Target Groups
- Auto Scaling Group
- AMI
- Security Groups
- CloudWatch
