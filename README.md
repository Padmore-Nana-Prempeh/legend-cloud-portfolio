# Legend Cloud Portfolio

Cloud Infrastructure • Data Engineering • Security Engineering • Observability • Automation

---
This repository documents a growing collection of production-inspired AWS implementations focused on secure cloud infrastructure, modern data engineering, operational monitoring, and event-driven automation.

Each project is designed, deployed, validated, and documented using AWS managed services while following architectural best practices, security principles, and reproducible implementation workflows.

Every project includes:

- Solution architecture
- Technical implementation
- Validation evidence
- Architecture diagrams
- Screenshots
- Technical documentation
